Что позволяет эта библиотека?

1.Простота расширения:
Чтобы добавить новую фигуру, достаточно создать новыйкласс, унаследованный от Shape,
реализовать метод "area()" и зарегистрировать класс в фабрике "create_shape".

2.Полиморфизм:
Независимо отконкретного класса, объекты фигур автоматически поддерживают единый
способ вычисления площади благодаря интерфейсу "Shape".
    - Клиентский код вызывает общий метод ".area()", не зная точный тип фигуры.
